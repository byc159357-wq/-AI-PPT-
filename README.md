# -AI-PPT-
一、项目简介  《智设工坊》是一款面向学生、设计师、新媒体运营和职场用户的 AI视觉内容生成软件。它可以根据用户输入的主题，自动完成 PPT大纲生成、页面排版、配图生成、照片编辑、海报设计和视觉风格统一，帮助用户快速完成汇报、作业、宣传图和作品展示。  简单说，它就是一个把 AI写作 + AI设计 + AI修图 + PPT排版 结合在一起的软件。  目前同类工具里，Gamma 可以用 AI 快速生成演示文稿并导出 PPT/PDF，Canva 也提供 AI 演示文稿和品牌化设计功能，Adobe Firefly 则主打 AI 图像生成与编辑，并正在强化跨 Adobe 软件的创意工作流。这个项目可以在这些方向上做一个综合型设计方案。
