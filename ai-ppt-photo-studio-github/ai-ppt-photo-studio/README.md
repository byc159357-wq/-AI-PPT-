# 智设工坊 / AI PPT Photo Studio

一个面向学生、设计师、新媒体运营和职场用户的 AI 视觉内容创作软件概念项目。  
它将 **AI PPT 生成、AI 照片创作、图片编辑、视觉风格分析和设计评审** 整合到一个工作流中，帮助用户快速完成汇报、作业、海报、视觉方案和作品集展示。

> 项目定位：AI 驱动的视觉创作工作台  
> 英文名：AI PPT Photo Studio  
> 中文名：智设工坊

---

## 项目简介

在日常学习和工作中，很多用户制作 PPT 或视觉图片时会遇到这些问题：

- 只有主题，但不知道如何拆解内容结构
- PPT 页面排版耗时长，风格不统一
- 图片素材来源杂乱，画面质感不稳定
- AI 生成图片容易出现“AI 感”、文字错误或构图混乱
- PPT、修图、海报、文案工具分散，工作流割裂

**智设工坊** 希望通过 AI 多 Agent 协作，将内容理解、页面策划、图片生成、照片编辑和设计评审整合起来，让用户从一个主题快速得到一套完整、可编辑、可优化的视觉作品。

---

## 核心功能

### 1. AI 一键生成 PPT

用户输入一个主题，例如：

```text
帮我做一个关于三孔桥信息可视化设计的前期调研 PPT
```

系统自动生成：

- PPT 标题
- 目录结构
- 页面大纲
- 每页文案
- 信息层级
- 版式建议
- 配图建议
- 图表类型建议
- 可导出 PPT / PDF / 图片

---

### 2. AI 照片与海报生成

用户可以输入提示词生成商业摄影、海报、插画或视觉概念图。

示例：

```text
高级香水广告海报，黑蓝渐变背景，玻璃瓶居中，电影级灯光，强反光，高级商业摄影风格
```

支持方向：

- 香水海报
- 产品摄影
- 城市写生
- 信息可视化背景图
- 动漫 IP 形象
- 武侠分镜画面
- 广告创意图

---

### 3. 图片编辑与优化

支持上传图片后进行 AI 编辑：

- 去除文字
- 更换背景
- 抠图
- 扩图成 16:9 / A4 / 海报尺寸
- 调整光影
- 修复透视
- 统一色调
- 生成同款不同版本
- 降低 AI 感

---

### 4. 风格参考系统

用户上传参考图后，系统自动分析：

- 色彩风格
- 构图方式
- 字体气质
- 光影关系
- 画面元素
- 设计关键词

然后将该风格应用到新的 PPT、图片或海报中。

---

### 5. AI 设计评审

系统会对生成结果进行二次审查，判断是否存在：

- 主题不突出
- 画面太满
- AI 感过强
- 字体不统一
- 色彩不协调
- 信息层级混乱
- 图片风格割裂
- 构图缺少重点

并自动给出修改建议。

## 多 Agent 核心逻辑

本项目采用多 Agent 协同工作流：

```text
用户输入主题 / 上传资料
        ↓
资料理解 Agent
        ↓
内容策划 Agent
        ↓
视觉生成 Agent
        ↓
设计评审 Agent
        ↓
输出 PPT / 图片 / 海报 / 作品集素材
```

### 资料理解 Agent

负责读取用户输入的主题、文档、PDF 或图片资料，提炼核心内容。

### 内容策划 Agent

负责生成 PPT 大纲、页面标题、正文层级和叙事逻辑。

### 视觉生成 Agent

负责生成图片、背景、图标、海报、插画和视觉素材。

### 设计评审 Agent

负责检查画面质量、设计统一性和 AI 生成痕迹，并提出优化建议。

## 项目亮点

### PPT 与图片一体化

传统流程中，PPT 制作、图片生成、修图和排版经常需要多个软件配合。  
本项目希望把这些能力整合进一个统一工作台。

### 面向设计学习场景

相比普通办公类 AI PPT 工具，本项目更关注视觉传达、广告设计、信息可视化和作品集展示。

### 降低 AI 感

通过设计评审 Agent 对画面进行二次判断，避免生成结果过于模板化、塑料感强或元素堆砌。

### 可扩展工作流

项目后续可以扩展为 Web 应用、桌面端工具、插件系统或本地 ComfyUI 工作流。

