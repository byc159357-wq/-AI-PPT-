# 产品需求文档 PRD

## 1. 用户群体

### 学生

需要完成设计作业、课程汇报、调研 PPT、作品集项目。

### 设计师

需要快速生成视觉方向、海报草图、风格参考和创意方案。

### 新媒体运营

需要快速制作推文封面、活动海报、产品宣传图。

### 职场用户

需要快速制作汇报 PPT、项目总结和方案展示。

---

## 2. 核心需求

### PPT 生成

- 输入主题后自动生成大纲
- 自动拆分页面结构
- 自动生成每页标题和正文
- 自动匹配配图和图表
- 自动统一字体、颜色和版式

### 图片生成

- 根据提示词生成图片
- 支持商业摄影、插画、写生、海报等风格
- 支持尺寸选择
- 支持多版本生成

### 图片编辑

- 去除文字
- 扩图
- 抠图
- 背景替换
- 光影修复
- 风格统一

### 设计评审

- 检测画面是否杂乱
- 检测信息层级是否清晰
- 检测是否存在 AI 感
- 生成修改建议

---

## 3. 功能优先级

| 优先级 | 功能 | 说明 |
|---|---|---|
| P0 | PPT 大纲生成 | 项目基础能力 |
| P0 | 页面文案生成 | 生成完整内容 |
| P1 | 图片提示词生成 | 用于生成视觉素材 |
| P1 | 设计评审 | 提高作品质量 |
| P2 | PPTX 导出 | 后续开发 |
| P2 | 图片编辑 | 后续增强 |
| P3 | Web 前端 | 进一步产品化 |

---

## 4. 输出格式

系统支持输出：

- Markdown 大纲
- PPT 页面脚本
- 图片生成提示词
- 海报文案
- 设计评审报告
- 后续可扩展为 PPTX 文件
