# Prompt 示例

## 1. PPT 生成 Prompt

```text
请帮我生成一个关于“AI 制作 PPT 与照片的软件”的项目展示 PPT。
要求包含：项目背景、用户痛点、核心功能、产品流程、多 Agent 逻辑、界面设计方向和未来规划。
整体风格高级、简洁、科技感，适合放入作品集。
```

---

## 2. 图片生成 Prompt

```text
AI 视觉创作软件界面概念图，深色科技风界面，中间是 PPT 页面编辑区，右侧是 AI 图片生成面板，左侧是项目资料栏，蓝紫色渐变光效，现代 UI 设计，高级感，清晰，干净，作品集展示风格
```

---

## 3. 海报生成 Prompt

```text
一张宣传 AI PPT Photo Studio 的科技感海报，画面中央是一台笔记本电脑，屏幕中显示自动生成的 PPT 页面和 AI 图片素材，周围漂浮着图表、照片、文字卡片和设计元素，深蓝紫渐变背景，未来感，商业科技广告风格
```

---

## 4. 设计评审 Prompt

```text
请作为专业视觉设计评审，检查下面这张 AI 生成海报是否存在 AI 感、构图混乱、字体不统一、主题不突出等问题，并给出具体修改建议。
```

---

## 5. 降低 AI 感 Prompt

```text
请优化这张图片，使其更像真实摄影作品，减少 AI 生成感。要求：自然光影、真实材质、减少过度锐化、避免塑料质感、构图更克制、色彩更高级。
```
