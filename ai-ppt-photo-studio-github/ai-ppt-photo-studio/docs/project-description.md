# 项目描述

## 项目名称

智设工坊 / AI PPT Photo Studio

## 项目类型

AI 视觉内容生成软件 / AI 设计辅助工具 / PPT 自动生成工具

## 项目背景

随着生成式 AI 的发展，用户可以通过文字快速生成图片、文案和演示内容。但在实际学习和工作中，用户仍然面临工作流割裂的问题：写 PPT 需要一个软件，生成图片需要一个平台，修图需要另一个工具，最终还要手动统一风格和排版。

本项目希望构建一个整合式 AI 创作工作台，帮助用户从“主题输入”直接进入“完整视觉方案输出”。

## 项目目标

- 降低 PPT 和图片创作门槛
- 提高设计作业和职场汇报效率
- 统一文字、图片、排版和视觉风格
- 帮助用户减少 AI 生成内容的廉价感和模板感
- 构建适合学生作品集展示的 AI 软件项目

## 核心痛点

1. 用户不知道如何从一个主题拆分出 PPT 结构
2. 设计排版耗时长，且效果不稳定
3. 图片素材风格不统一
4. AI 生成图容易出现画面杂乱、塑料感、文字错误等问题
5. 多软件切换导致效率低

## 解决方案

通过多 Agent 协作，将资料理解、内容策划、视觉生成、设计评审整合为完整工作流。

最终用户可以通过一句话生成 PPT 或图片方案，也可以上传资料，让系统自动提炼信息并生成视觉内容。
